# Directory Size Calculator

## Download JAR
[Here](https://github.com/DiegoB2003/Directory-Size-Calculator/releases/tag/v1.0.0)

## Requirements
- Java 8 or higher

## How to Run
```bash
java -jar DirectorySizeCalculator.jar
```

## Commands
- cd \<directory name\> OR .. (goes to parent directory of current directory)
- ls (lists all files and directories in current directory)
- size (recursively calculates total size of all children from the current directory)
